## 헤더
html의 `h1` 또는 `h2`와 같은 헤더 태그는 `#`으로 표시할 수 있다.

```
# 이것은 h1
## 이것은 h2
### 이것은 h3
#### 이것은 h4
##### 이것은 h5
###### 이것은 h6
```

## code snippet
코드를 넣을 때는 아래와 같이 입력한다.
{% raw %}
```
{% highlight javascript %}
/* this is javascript comment */
var raw = ["r", "a", "w"];
{% endhighlight %}
```
{% endraw %}

## 이탤릭, 볼드, 선긋기

```
*이탤릭*

**볼드**
__이것도 볼드__
~~선긋기~~
```
